# Code Blocks

DubStack provides powerful code block features with syntax highlighting, line numbers, and more.

## Basic Code Block

Use triple backticks with a language identifier:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Code Block Features

### With Title

```python hello.py
def hello():
    print("Hello, World!")
```

### With Line Numbers

```javascript showLineNumbers
const x = 1;
const y = 2;
const sum = x + y;
console.log(sum);
```

### With Line Highlighting

```typescript {2-3}
function calculate() {
  const result = 42;
  return result;
}
```
