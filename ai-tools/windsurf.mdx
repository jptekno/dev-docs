# Windsurf Integration

Windsurf is an AI-native IDE with powerful documentation editing capabilities.

## Getting Started

<Steps>
  <Step title="Install Windsurf">
    Download from [codeium.com/windsurf](https://codeium.com/windsurf)
  </Step>

  <Step title="Open your docs folder">
    Open your documentation repository in Windsurf.
  </Step>

  <Step title="Configure workspace rules">
    Create a `.windsurf/rules` directory for custom AI instructions.
  </Step>
</Steps>

## Workspace Rules

Create documentation-specific rules for better AI assistance:

```markdown .windsurf/rules/docs.md
# Documentation Rules

## Writing Style
- Use second person ("you")
- Keep sentences concise
- Include code examples

## Components
- Use <Note> for tips
- Use <Warning> for cautions
- Use <Steps> for procedures

## Structure
- Start with frontmatter
- Use descriptive headings
- Include next steps
```
