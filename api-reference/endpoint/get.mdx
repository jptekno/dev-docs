# Get User

Retrieve detailed information about a specific user.

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user
</ParamField>

## Query Parameters

<ParamField query="include" type="string">
  Comma-separated list of related resources to include. Options: `profile`, `settings`, `teams`
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique user identifier
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="name" type="string">
  User's display name
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the user was created
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/v1/users/usr_123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/users/usr_123', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const user = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/v1/users/usr_123',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)
user = response.json()
```
</RequestExample>
